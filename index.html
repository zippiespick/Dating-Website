Description of Project Languages, Tools, Frameworks Used

This project was worked on for my Senior project throughout the Summer and Fall of 2020. I mainly contributed to the group feature and the StudentsSection page of the website. The group feature is for the purpose of dividing the students in groups for each of their classes. These groups would then have assignments that can help each other boost the number of points throughout the course. The StudentsSection page demonstrates the list of students in the course and the groups that they are a part of. The admin can see all of the groups of a course whereas the students can see their own group in a course. I also worked on the progression feature for courses, specifically PDFs, where students complete and earn points as they finish assignments. The website can be found in this link.

Game Development

Unbound

Description of Project Languages, Tools, Frameworks Used

This project serves to be a personal hobby. It is a Unity first person shooter game where the player is trying to survive from alien-like creatures in a forest. As you can see, it is still a work in progress but the features that have been implemented thus far include the UI, weapon swapping, enemy behavior, and a collision system between the enemies and the player. Me and a friend of mine have been looking to improve this project in terms of animations, enemy variety, debugging, and level design. Our main objective at the moment is to complete the first level with the gameplay systems that have been and will be set in place.
